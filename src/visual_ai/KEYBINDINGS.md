# Slay the Spire 2 快捷键

用于 AI 控制的快捷键列表。

## 战斗界面

| 快捷键 | 功能 | 说明 |
|--------|------|------|
| `1-9` | 查看手牌 | 按下数字键查看对应位置的卡牌详细信息 |
| `1-9` | 选择目标 | 当卡牌需要选择目标时，按数字选择敌人 |
| `Space` | 结束回合 | 结束当前回合 |
| `Z` | 撤销/返回 | 撤销上一个操作或返回 |
| `Tab` | 查看卡牌 | 查看当前选中的卡牌详情 |
| `P` | 暂停游戏 | 暂停游戏 |

## 地图界面

| 快捷键 | 功能 | 说明 |
|--------|------|------|
| `WASD` / 方向键 | 移动地图 | 在地图上移动选择路径 |
| `Enter` / `Space` | 确认选择 | 选择当前高亮的路径 |
| `1-5` | 快速选择 | 快速选择对应编号的路径 |

## 商店界面

| 快捷键 | 功能 | 说明 |
|--------|------|------|
| `1-5` | 选择商品 | 选择对应编号的商品 |
| `Enter` | 购买 | 购买当前选中的商品 |
| `Tab` | 切换标签页 | 在商店/移除/升级之间切换 |
| `Q` | 退出商店 | 离开商店 |

## 事件界面

| 快捷键 | 功能 | 说明 |
|--------|------|------|
| `1-3` | 选择选项 | 选择对应编号的事件选项 |
| `Enter` | 确认 | 确认当前选项 |
| `Space` | 跳过 | 跳过当前事件（如果有选项） |

## 卡牌奖励界面

| 快捷键 | 功能 | 说明 |
|--------|------|------|
| `1-3` | 选择卡牌 | 选择对应编号的卡牌奖励 |
| `Enter` | 确认 | 确认选择 |
| `Z` | 跳过 | 跳过卡牌奖励 |

## 休息界面

| 快捷键 | 功能 | 说明 |
|--------|------|------|
| `1-3` | 选择操作 | 选择：1=休息，2=升级，3=回忆 |
| `Enter` | 确认 | 确认当前操作 |

## 通用快捷键

| 快捷键 | 功能 | 说明 |
|--------|------|------|
| `Esc` | 返回/取消 | 返回上一级菜单或取消当前操作 |
| `Tab` | 切换标签 | 在游戏内不同标签页之间切换 |
| `F1` | 查看帮助 | 查看游戏帮助（如有） |
| `Ctrl+Z` | 撤销 | 撤销上一个操作 |

## AI 实现策略

### 方案 1：纯快捷键控制

使用数字键快速选择和执行操作：

```python
# 打第 3 张牌
keyboard.press('3')
time.sleep(0.1)

# 选择第 1 个目标敌人
keyboard.press('1')
time.sleep(0.1)

# 结束回合
keyboard.press('space')
```

### 方案 2：混合方案（主要用鼠标 + 快捷键辅助）

- 卡牌定位用鼠标（更精确）
- 快捷键用于确认、取消、跳过等通用操作
- 目标选择用数字键（更快）

## 注意事项

1. **数字键对应** - 游戏中的数字键通常对应从左到右或从上到下的顺序
2. **键盘布局** - 确保游戏使用的键盘布局与系统一致
3. **延迟时间** - AI 操作后需要适当延迟，等待游戏响应
4. **状态同步** - 在执行操作前确认游戏处于正确的界面和状态

## 配置方法

如需修改快捷键，在游戏中：
1. 按 `P` 键打开暂停菜单
2. 进入 "设置" → "控制"
3. 修改对应快捷键
4. 在 AI 代码中同步更新

## 示例实现

```python
class KeyboardController:
    def play_card(self, card_index: int, target_index: int = -1):
        """使用快捷键打出卡牌"""
        # 选择手牌位置（如果游戏支持）
        # STS2 可能不需要先选手牌，直接点击即可

        if target_index >= 0:
            # 选择目标敌人
            self.press(str(target_index + 1))
            time.sleep(0.1)

    def end_turn(self):
        """结束回合"""
        self.press('space')
        time.sleep(0.2)

    def select_reward(self, card_index: int):
        """选择卡牌奖励"""
        self.press(str(card_index + 1))
        time.sleep(0.1)
        self.press('enter')
```
