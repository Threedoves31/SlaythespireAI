# 底层方案
将mod文件放入mods文件夹即可

# 编译方法
双击install_mod.bat:
- __编译__ - `dotnet build` Debug 配置

- __创建目录__ - `D:\Steam\steamapps\common\Slay the Spire 2\mods\STS2AIBot`

- __复制文件__：

  - ✅ `STS2AIBot.dll` - 主程序
  - ✅ `STS2AIBot.pdb` - 调试符号
  - ✅ `STS2AIBot.pck` - 资源包
  - ✅ `modinfo.json` - Mod 信息
  - ✅ `mod_manifest.json` - 清单文件
